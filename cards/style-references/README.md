# Style References

Drop 2-4 exemplar images here that define the card art style. Every
generation run attaches all images in this directory after the employee's
reference photos, and the prompt tells the model to copy their artistic
style — palette, rendering technique, background treatment — without
borrowing faces or identity.

Good style references:

- Finished cards from an earlier batch, so new cards match the existing set.
- Commissioned or hand-picked artwork in the target style.
- Consistent with each other; conflicting styles produce muddy results.

Supported formats: PNG, JPG, WEBP.
