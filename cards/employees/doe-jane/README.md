# Jane Doe — Reference Photos

This is an example employee folder. Put 1-3 clear reference photos of the
employee in this directory (PNG, JPG, or WEBP). The pipeline auto-discovers
them and uses them only for likeness — facial structure, hair, glasses,
build — never as the final art.

Good reference photos:

- Well lit, front-facing or three-quarter view.
- Recent, so the art matches how the person looks today.
- Free of sunglasses, heavy filters, or other people in frame.

Get each employee's consent before adding their photos, and prefer keeping
photos out of version control if the repository is shared (see
`cards/README.md`).
